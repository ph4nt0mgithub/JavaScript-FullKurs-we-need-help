# Logical Operators:

| Operators | Description                               | Example                                      |
|-----------|-------------------------------------------|----------------------------------------------|
|    &&     | Logical AND                               | `let andResult = (true && false); // andResult is false` |
|    \|\|     | Logical OR                                | `let orResult = (true \|\| false); // orResult is true` |
|    !      | Logical NOT                               | `let notResult = !true; // notResult is false` |
